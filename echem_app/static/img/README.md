# Logo files

Drop the two PNG files here so the app can render them in the sidebar.
The app will fall back to text labels if the files are missing.

| File | Description | Where it appears |
|---|---|---|
| `neotrient_logo.png` | Neotrient brand mark — periwinkle "Neotrient" text + droplet/leaf | Top of the sidebar |
| `abic_logo.png`      | Advanced Biomedical Instrumentation Centre / 先進生物醫學儀器中心 | "Supported by" block at the bottom of the sidebar |

Recommended size: anything from 256–1024 px wide. The CSS scales them
automatically (`max-width: 180px` for the brand, `max-height: 56px` for
the sponsor).
