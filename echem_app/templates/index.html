User guide

Six phases per sample.

  1. Configuration — operator, device, preset
  2. Sample identification — scan QR
  3. Sample loading — place the sample
  4. Measurement — wait for CV to complete
  5. Analysis — review predictions
  6. Save data — sign and export

Pre-measurement cautions

  • Polish the working electrode before each sample.
  • Verify no air bubbles on the electrode surface.
  • If primary model and linear baseline disagree by > 30%, an override justification is required.
  • Manual overrides are recorded in the audit log.
  • Allow camera access on first launch.
